ASSETS FOLDER
=============

This folder contains all game assets organized by type.

CONTENTS:
- 3D_Models: Car models, environment models, and other 3D assets
  - Car_Models: Original car model collection
  - GLB_Models: 3D.CARS GLB files for web/game use
- Textures: All texture files for materials and surfaces
  - Original_Textures: Your original texture collection
- Audio: Sound effects, music, and voice files
- UI_Graphics: User interface graphics and icons

PURPOSE:
Centralized asset management for easy access during development and to prevent duplicate files.
